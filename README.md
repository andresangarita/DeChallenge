# Installation

1. create a virtual env 
```[python]
python -m ven env 
```

2. activate the venv

in windows
```
env\Scripts\activate.bat
```
in linux
```
source env/bin/activate
```
3. install dependencies
```
pip install -r requirements.txt
```
